## How to contribute a translation
Language contributions are welcome!

**How to submit a new translation:**

1. Fork the repo
1. Create a resource file for the language you want to translate.
1. Translate it (obviously)
1. Add the new language to the `supportedCultures` list in `Startup.cs`.
1. Open a pull request

I will do my best to integrates PR as fast as possible.
